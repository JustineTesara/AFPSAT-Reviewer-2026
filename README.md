### ğŸ¯ **Section-Specific Tips**

#### **Verbal Reasoning**

**Vocabulary:**

- Learn 10 new words daily
- Use flashcards or apps
- Read quality materials (newspapers, articles)
- Practice using words in sentences

**Grammar:**

- Review basic rules (subject-verb agreement, tenses)
- Practice error identification
- Read correct grammar examples

**Analogies:**

- Understand relationship types (synonym, antonym, part-to-whole)
- Practice identifying patterns
- Think of bridge sentences: "A is to B as C is to D"

**Time Strategy:**

- Easy questions: 15-20 seconds
- Medium questions: 25-30 seconds
- Hard questions: Skip if over 35 seconds, come back if time

---

#### **Numerical Reasoning**

**Mental Math:**

- Memorize multiplication tables (1-12)
- Learn quick calculation tricks
- Practice estimation
- Time yourself daily

**Word Problems:**

- Read carefully, identify what's being asked
- Draw diagrams if helpful
- Write down the formula/equation
- Double-check units and final answer

**Key Topics to Master:**

- Percentages (very common!)
- Ratios and proportions
- Basic algebra
- Fractions and decimals
- Average, speed, distance, time

**Time Strategy:**

- Quick scan: 5 seconds (can you solve it fast?)
- Simple problems: 30-40 seconds
- Complex problems: 50-60 seconds
- Skip and return strategy: Mark difficult ones, solve easy ones first

---

#### **Abstract Reasoning**

**Pattern Types:**

- Rotation: clockwise/counterclockwise
- Progression: shapes increasing/decreasing
- Alternation: switching between 2-3 states
- Addition/subtraction: elements being added/removed
